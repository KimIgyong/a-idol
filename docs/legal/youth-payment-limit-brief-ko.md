## 1. 요청 개요

당사는 **"A-idol"**이라는 AI 아이돌 팬덤 모바일 서비스 (iOS/Android,
이하 "서비스")의 MVP 출시를 **2026-08-29**로 목표하고 있습니다.

서비스는 가상 아이돌 캐릭터와의 **채팅**, **오디션 라운드 투표**,
**포토카드 뽑기(가챠)** 기능을 유료 상품과 결합해 제공합니다. 모든
유료 상품은 Apple IAP / Google Play Billing을 통해 결제될 예정입니다.

출시 전 **다음 다섯 가지 질의**에 대한 서면 의견이 필요합니다:

1. 이 서비스가 **「게임산업진흥에 관한 법률」 제2조의 "게임물"**
   정의에 해당하는가?
2. 해당한다면, **청소년 월 결제 한도 5만원**(게임물관리위원회 청소년
   이용가 등급 고시상 한도)이 적용되는가? 적용 범위와 합산 단위는?
3. 해당하지 않는다면, **어떤 법규가 대체 적용**되는가? 전자상거래법
   표시·광고·환불 의무, 청소년보호법 일반 조항(유해정보) 등과의
   관계는?
4. 서비스 내 세 유료 상품(**채팅 쿠폰 · 오디션 투표권 · 포토카드
   뽑기 팩**)이 서로 다른 법적 분류를 받을 가능성이 있는가?
5. **국내 출시 후 중국·동남아 이용자가 VPN 등으로 접속해 결제**할
   경우, 국내법(특히 4항 분류) 적용 범위 및 IP 차단 의무 여부?

## 2. 서비스 개요

### 2.1 제품 구성

| 영역 | 설명 | 유료 상품 여부 |
|---|---|---|
| 아이돌 카탈로그 | 99명의 가상 아이돌 프로필·사진·일정 열람 | 무료 |
| 하트·팔로우 | 아이돌별 좋아요/팔로우 | 무료 |
| 팬클럽 가입 | 각 아이돌별 무료 커뮤니티 가입 | 무료 (유료 팬클럽은 MVP 제외) |
| **AI 채팅** | 아이돌 캐릭터와 1:1 대화. 하루 무료 메시지 제공, 초과 시 쿠폰 차감 | **채팅 쿠폰 구매 가능** |
| **오디션 투표** | 관리자가 지정한 아이돌 라운드에서 투표 | **투표권 구매 가능** |
| **포토카드** | 아이돌별 포토카드 세트. 팩 구매 시 가중치 기반 랜덤 롤 | **포토카드 팩 구매 가능** |

### 2.2 상품 설계 상세 (법적 분류 판단용)

#### (a) 채팅 쿠폰 (`CHAT_COUPON`)
- 기능: 사용자의 하루 무료 메시지(현재 기본 10건) 초과 시 1 메시지 = 1 쿠폰 차감
- 가격 예시: 10매 1,100원 / 30매 3,000원 / 100매 8,900원
- 랜덤성: 없음. 1매 = 1 메시지, 확정적 소비.
- 환전성: 없음. 앱 내 채팅 기능 소비 외 용도 없음.

#### (b) 오디션 투표권 (`VOTE_TICKET`)
- 기능: 진행 중인 오디션 라운드에서 아이돌에 투표 시 차감
- 두 유형:
  - 글로벌 투표권: 어느 라운드에서든 사용 가능
  - 라운드 전용 투표권: 특정 라운드에서만 유효, 해당 라운드 종료 시 사용 불가
- 가격 예시: 10매 1,100원 / 50매 4,900원 / 라운드 전용 프로모션은 할인가
- 랜덤성: 없음. 투표 가중치(`ticketWeight`)는 라운드 별 관리자가 사전 고지한 고정값.
- 환전성: 없음. 라운드 결과는 UI 랭킹 표시 외 혜택 없음(경품 없음).

#### (c) 포토카드 팩 (`PHOTOCARD_PACK`)
- 기능: 팩 구매 시 지정된 세트에서 N장의 포토카드가 **가중치 기반 랜덤**으로 사용자 인벤토리에 추가됨
- 가격 예시: 5장 팩 2,500원 / 10장 팩 4,500원
- 랜덤성: **있음**. 각 템플릿의 `dropWeight`로 확률 결정.
- **확률 공개**: 상점 화면에 각 카드별 획득 확률 % 표시 (서버 계산,
  조작 불가) — 첨부 ADR-016 참조.
- 환전성: **없음** (의도된 정책, ADR-018 참조) — 카드 간 교환·선물·
  환전·소각·외부 판매 전 기능 미구현. 관리자 수동 재지급(버그 대응)
  만 가능.
- 가치: 카드는 앱 내 컬렉션에 표시되며, 경품·상금·외부 혜택과 연결
  되지 않음.

### 2.3 유저 유입

- 대상: 한국 20~30대 K-pop 팬덤 중심, 청소년 접근 가능성 높음.
- 회원가입: 이메일 + 비밀번호 + **생년월일 필수** (만 14세 미만
  가입 차단 예정, 세부 정책 5.1절 참조).
- 결제: Apple IAP / Google Play Billing을 통해서만 (MVP 출시 이후).

## 3. 법적 쟁점

### 3.1 "게임물" 해당 여부 (질의 1)

게임산업법 제2조 제1호에 따른 "게임물" 정의:
> "정보통신기술을 이용하거나 기계·전자장치 등을 이용하여 오락을 할 수
> 있게 하거나 이에 부수하여 여가선용·학습 및 운동효과 등을 높일 수
> 있도록 제작된 영상물 또는 그 영상물의 이용을 주된 목적으로 제작된
> 기기 및 장치를 말한다."

**우리 서비스의 위치**:
- 오락성: 채팅·투표·포토카드 수집 모두 오락 요소. **긍정 방향**.
- 승패·점수: 오디션 투표는 랭킹이 있으나 사용자가 "이기는" 개념은
  없음(참여 아이돌의 랭킹). 점수는 부수적.
- 랜덤 결과물: 포토카드 가챠가 존재. 다른 게임과 유사.
- 구별점: 주요 기능(아이돌 카탈로그·채팅·팬덤)은 "게임"보다 **팬덤
  커뮤니티/미디어**에 가까움. 전통적 게임의 승패 메커닉 부재.

**우리의 가설적 입장**: 서비스 전체는 팬덤 플랫폼이며, 포토카드 뽑기
기능만 부분적으로 게임성 요소를 포함함. 단 법적 판단은 법무의 의견
요망.

### 3.2 청소년 월 결제 한도 5만원 적용 (질의 2)

게임물관리위원회 고시상 **청소년 이용가 게임물의 월 결제 한도 5만원**
(아동·청소년 = 만 18세 미만, 게임산업법 2조 제10호)이 적용되는 경우:

- **상품별 vs 전체 합산** 해석? 포토카드만 게임물이라면 포토카드
  월 한도만 5만원인지, 아니면 서비스 전체 유료 결제 합산 5만원인지.
- **합산 단위**: 앱 단위? 퍼블리셔 단위? 계정 단위?
- **본인인증·KYC**: 청소년 식별 방법. 생년월일 자진 신고 + 법정대리인
  동의로 충분한가, 아니면 실명인증 필수인가?
- **위반 시 제재**: 한도 초과 결제가 발생한 경우 의무적으로 환불해야
  하는 범위.

### 3.3 환전 금지 (§32.1.7) — 이미 대응함 (ADR-018 참조)

당사는 포토카드의 사용자 간 교환·선물·소각·환전 일체를 MVP에서
차단하는 것으로 **내부 결정 완료** (ADR-018). 이 결정 자체에 대한
법무 검토도 함께 요청드립니다 — 추가 보강 포인트가 있는지.

### 3.4 확률 공개 (§22 준수) — 이미 대응함 (ADR-016 참조)

2024-03-22 시행된 확률 공개 의무는 포토카드 팩의 상점 화면에 각 카드별
확률을 서버 계산값으로 공개해 대응 중. 조문 해석상 공개 위치/형식이
충분한지 확인 요청.

### 3.5 그 외 법규 (질의 3·4·5)

- **전자상거래법**: 청약 철회 · 환불 규정 적용 여부. 가챠 결과물의
  "비가역적 디지털 상품" 예외 해당 여부.
- **청소년보호법**: 유해정보 차단 의무 (채팅 내용). AI 응답이 부적
  절 발언을 생성할 경우 책임 범위.
- **특정금융정보법**: 포토카드를 가상자산으로 간주할 여지. 당사 입장은
  "아님" (§2.2.c 환전성 미제공으로 해명).
- **위치정보법 / 개인정보보호법**: 가입 시 수집 항목. 표준 개인정보
  처리방침 템플릿 확인 요청.
- **해외 이용자**: 국내법 적용 범위 · VPN 접속자 판단 기준 · 해외
  퍼블리싱 시 필요한 계약 구조 (별도 법인? 개별 라이선스?).

## 5. 당사의 가설적 입장 (법무 검토 출발점)

**의견 요청 시 참고**하시라고 밝히는 당사 입장이며, 법적 판단 아닙니다.

### 5.1 청소년 보호 (보수적 기본값)

**현재 MVP에 선탑재 예정**:
- 만 14세 미만 가입 차단 (`UNDER_AGE` 에러 코드 이미 구현, 회원가입
  시 생년월일 요구).
- 만 14세 이상~18세 미만: 결제 화면에서 월 결제 누적 5만원 초과 시
  경고 후 차단 (법무 의견 확정 후 on/off 전환 가능한 feature flag로 구현).
- 법정대리인 동의 플로우는 **미구현** — 법무 의견에 따라 추가.

### 5.2 서비스 분류

팬덤 플랫폼 전체는 게임물 비해당, 포토카드 뽑기 기능만 "게임물 요소"
포함의 가능성. 다만 법적 판단은 법무 의견 요망.

### 5.3 환전성 차단

§32.1.7 대응으로 **모든 사용자간 자원 이동 금지** (ADR-018). 환불은
시스템 오류 케이스로 한정.

### 5.4 확률 공개

게임산업법 §22 + Apple §3.1.1 + Google Play policy 삼중 대응. 서버
계산 %로 조작 불가.

## 부록 A — 법정 대리인 동의 UI 설계 옵션 (참고)

법무 의견에 "법정대리인 동의 필요" 확정 시 구현할 UI 옵션:

1. **회원가입 시 생년월일로 만 14세 이상 18세 미만 판별 → 법정대리
   인 이메일 입력 → 이메일 링크 승인** — 표준 방식.
2. **이동통신 실명인증 (NICE, KCB)** — 신분확실성 높지만 가입 이탈률
   상승 우려.
3. **간편인증 (카카오/네이버)** — 접근성 좋으나 14세 미만/법정대리인
   구분 난이도.

법무 권고 수신 후 1주일 내 UI 확정 가능.
